# Celullar Automatas 

A simple implementation of celular automatas to experiment with raylib´s graphic interface.

## Dependencies
To compile this project you need:
- A C compiler (gcc)
- Raylib librarie version 5.0 or higher

## Installing raylib

**Windows**
You can install it with [Raylib´s](https://github.com/raysan5/raylib/releases) official installer

**Linux**
```bash
sudo apt install libraylib-dev
```
## Using the program

You can compile and execute it using `make run`.

Then you may click some squares to set an initial state for the grid. After that you can start updating the generations!!
